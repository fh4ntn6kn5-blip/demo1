# 智能知识库文档编辑器

基于 Feishu/Lark 风格的现代知识库文档编辑器原型，支持 @ 引用与 Sugar AI 智能体。

## 功能特性

- **双栏布局**：可折叠侧边栏（约 260px）+ 主编辑区域
- **树状目录**：知识库文档树、最近访问、收藏
- **@ 唤起机制**：输入 `@` 弹出引用菜单，首项为 Sugar 智能体
- **Sugar AI 模式**：选择 Sugar 后进入 AI 对话/指令框，带紫色发光效果
- **数据列表**：在 Sugar 中输入并确认后，文档下方展示固定数据列表

## 技术栈

- React 18
- Tailwind CSS
- Lucide React（图标）
- Framer Motion（动画）

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建
npm run build
```

## 使用流程

1. 在编辑器中输入 `@` → 弹出引用悬浮菜单
2. 选择「Sugar 智能体」（紫色 AI 图标）→ 进入 AI 模式
3. 输入问题或续写内容 → 点击「确认」
4. 文档下方展示固定数据列表
